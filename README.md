# Codex 分享资料包

这个资料包是给 Codex 读取的视频总结材料。

## 文件说明

- `docs/persuasion_three_elements_lazy_brain.md`：视频总结的 Markdown 版本，最推荐给 Codex 读取。
- `docs/persuasion_three_elements_lazy_brain.txt`：纯文本备份版本。
- `AGENTS.md`：放在项目根目录后，Codex 会把它当作项目规则/上下文参考。
- `PROMPT_FOR_CODEX.md`：可直接复制到 Codex 的提示词。

## 推荐用法

1. 把本资料包里的所有文件复制到你的 GitHub 项目根目录。
2. 打开 Codex，让它读取项目。
3. 把 `PROMPT_FOR_CODEX.md` 里的内容复制给 Codex。
4. 让 Codex 根据 `docs/persuasion_three_elements_lazy_brain.md` 开始设计或写代码。

## 最简单的一句话提示

请先阅读 `docs/persuasion_three_elements_lazy_brain.md`，然后基于里面的内容帮我做一个说服力训练工具/课程笔记整理工具/AI 法律助手功能原型。
